# BLACK-JACK

A Blackjack game built with **JavaFX**, **OOP principles**, and **MySQL database integration**.
This project demonstrates use of **Object-Oriented Programming**, **GUI design**, **data persistence**, and **game logic implementation**.

---

## Features

* User **Login / Registration** system
* Persistent user data using **MySQL**
* Blackjack rules implemented:

  * Hit / Stand
  * Double Down
  * Split
  * Surrender
  * Insurance
* Betting system
* User profile management:
  
  * Change username / password
  * View game history
  * View stats (wins, losses, win rate, credits)
* Background music and sound effects
* Settings screen for audio control
* Game guide overlay
* Multi-screen JavaFX interface
